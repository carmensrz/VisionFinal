Proyecto final de Visión Computacional
======================================

Proyecto final de la materia de visión computacional, el cual es un proyecto que realiza detección de objetos mediante
métodos de visión computacional, con el objetivo de encontrar en las imágenes características similares que puedan
hacer que se llegue a la conclusión de que en las dos imágenes se encuentra el mismo objeto.

Instalación
===========
Leer el archivo INSTALL para conocer las librerías que tienen que instalarse para poder correr este programa.

Ejecución
=========
Para ejecutar el programa desde ubuntu se habre la terminal y se pone python seguido del nombre del archivo y 
seguido de las dos imágenes que se van a tratar.


Ejemplo:
python visionfinal.py image1.jpg image2.jpg


Uso
===
En el repositorio se encuentran algunas de las imágenes que utilice pero se puede cambiar a cualquier otra imagen
si se requiere.
